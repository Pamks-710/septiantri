# Sistem CRUD Data Mahasiswa – Ujian RPL2
Proyek ini dibuat untuk memenuhi Ujian Akhir Mata Kuliah Rekayasa Perangkat Lunak 2. Aplikasi ini merupakan sistem CRUD (Create, Read, Update, Delete) sederhana berbasis PHP dan MySQL untuk mengelola data mahasiswa.

## Fitur
1. Tambah data mahasiswa
2. Edit data mahasiswa
3. Hapus data mahasiswa
4. Tampilkan data mahasiswa

## Teknologi
1. PHP
2. MySQL
3. Bootstrap
4. Composer
5. Git & GitHub

## Struktur Folder
1. models/ – File CRUD  
2. vendor/ – Dependency Composer  
3. config.php – Koneksi DB  
4. create_table.sql – Struktur tabel  
5. index.php – Halaman utama  
6. edit.php – Edit data  
7. delete.php – Hapus data  

## Cara Menjalankan
1. Pindahkan folder proyek ke: xampp/htdocs/
2. Buat database: rpl2_act4
3. Import create_table.sql
4. Jalankan Apache & MySQL
5. Akses aplikasi: http://localhost/rpl2_act4_php/
