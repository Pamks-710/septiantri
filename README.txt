RPL2 ACT4 – Starter PHP (XAMPP)
Langkah cepat (≤ 20 menit):
1) XAMPP: Start Apache & MySQL.
2) Pastikan DB rpl2_act4 sudah ada. Jika tabel belum ada, jalankan create_table.sql.
3) Copy folder ini ke htdocs: C:\xampp\htdocs\rpl2_act4_php
4) Buka http://localhost/rpl2_act4_php
5) Tambah data via form → data tampil di tabel.
6) Ambil screenshot (form+tabel), kode db.php & index.php, dan phpMyAdmin.
